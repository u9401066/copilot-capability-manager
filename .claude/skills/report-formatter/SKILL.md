# 報告整理技能

## 描述
對已產出的報告進行格式化、美化和內容優化。

## 觸發條件
- 「整理報告」
- 「格式化報告」
- 「美化報告格式」
- 報告產出後的後處理

## 處理項目

### 格式優化
- [ ] 統一標題層級
- [ ] 修正 Markdown 語法
- [ ] 添加適當的空行和分隔
- [ ] 統一列表格式 (有序/無序)

### 內容優化
- [ ] 檢查文法錯誤
- [ ] 統一術語用法
- [ ] 優化段落結構
- [ ] 確保邏輯連貫

### 視覺優化
- [ ] 添加適當的 emoji 標記
- [ ] 使用表格整理數據
- [ ] 添加程式碼區塊高亮
- [ ] 適當使用粗體/斜體強調

## 輸出格式
```
📋 報告整理完成

✅ 格式修正：N 處
✅ 文法修正：N 處
✅ 結構優化：N 處

變更摘要：
- [具體變更 1]
- [具體變更 2]

報告已儲存至：[路徑]
```

## 使用範例
```
「整理剛才的報告」
「格式化報告為正式版本」
「美化這份報告的排版」
